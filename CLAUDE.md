# MCP プロジェクト

## Skills

### /genba $SITE_NAME
Description: 建築現場の情報を検索する
Use the `search_construction_site` MCP tool with site_name set to "$SITE_NAME" to search for construction site information. Display the results in a readable format.

### /genba-companies $SITE_NAME
Description: 建築現場の関連会社を検索する
Use the `search_site_companies` MCP tool with site_name set to "$SITE_NAME" to search for companies involved in the construction site. Display the results in a readable format.

### /genba-config
Description: Gemini API の設定状態を確認する
Use the `get_gemini_config` MCP tool to check the current Gemini API configuration status.
